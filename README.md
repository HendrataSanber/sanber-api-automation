## Base URL

  https://kasir-api.zelz.my.id

## API Endpoints

  Create Category

    POST /categories

  Get Category By Id

    GET /categories/{categoryId}

  Update Category

    PUT /categories/{categoryId}

  Delete Category

    DELETE /categories/{categoryId}
